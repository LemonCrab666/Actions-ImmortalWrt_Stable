# Actions-OpenWrt

使用[P3TERX/Actions-OpenWrt](https://github.com/P3TERX/Actions-OpenWrt)的模板对lean's lede源码进行编译

## 纯净版X86_64 OpenWrt
已包含插件wireguard、ttyd终端、上网时间控制、openclash、全能推送、动态dns、网络唤醒、upnp、openvpn服务器、带宽监控
